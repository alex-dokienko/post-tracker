> Декларации с СМС больше не подгружаются (!!!)

К сожалению в связи с изменениями правил размещения приложений на Play Маркет данное приложение не имеет право использовать доступ к СМС.

В связи с чем пришлось данную функцию удалить во избежания удаления с Play Маркет.

> Как автоматически подгружать декларации в приложение (как в НП)

Данная возможность реализована в PRO версии.

1. Для этого нужно создать API key в электронном кабинете Новой Почты (более детально по ссылке):

  • в браузере зайти на сайт НП
  • ввести свой логин/пароль
  • перейти в меню Настройки->Api 2.0
  • нажать кнопку "создать новый ключ"
  • скопировать созданный ключ (пример ключа bfce3e492873e...)

2. Перейти в приложении на страницу Личного кабинта

3. Вставить в поле API token и сохранить

4. Нажмите кнопку обновить в приложении после чего будут загружены декларации за последних 2 недели

Обратите внимание:

  • деклараций созданные в отделении к сожалению не будут подгружаться таким способом.
  • созданный ключ имеет срок действия 1 год, не забудьте его обновить по истечению этого срока.

> Статусы не обновляются автоматически

Часто эта проблема связана с тем что приложению запрещена работа в фоне. Необходимо проверить что в настройках телефона выданы разрешение на работу в фоне.

Еще одной причиной при обновлении может быть отключенная глобальная синхронизация, более детально можно прочитать по ссылке

> Как переустановить приложение не потеряв данные

1. Перед удалением приложения необходимо воспользоватся опцией меню Настройки=>Архивировать, после чего выбрать место в памяти телефона для сохранения файла

2. Удалить приложение и снова установить приложение используя стандартное приложение Play Store

3. Зайти снова в меню настроек приложения и выбрать меню Настройки=>Восстановить, выбрать из памяти телефона сохранненый ранее файл (пример названия файла post-tracker-2018-02-18_10-59.db)

> Функция отправки СМС подставляет не правильно номер телефона

Для того что бы правильно работала эта функция необходимо обязательно при добавлении декларации вводить свой номер телефона в качестве номера для отслеживания.

Номер для отслеживания можно изменить даже после того как декларация была добавлена в приложение, для этого в деталях декларации нужно просто нажать на этот номер.

> Преимущества PRO версии

При установке платной версии приложения, вы получаете:

- возможность ввести Карту Клиента и предоставлять щтрих-код в отделении при получении декларации.

- возможность автоматического добавления деклараций предоставив свой API key, созданных через эл. кабинет/приложение (кроме деклараций созданных в отделении).

- режим "Сканирование-Добавление-Отправка СМС" для более быстрого занесения деклараций.

- подсчет суммы обратных декларации для выбранных деклараций.

- формирование по шаблону информации о посылке в поле описание товара.

- оперативное исправление ошибок, добавление необходимых функций
